# GlassFish Template

## Requirements

- OpenJDK 21.0.5 2024-10-15 LTS (Temurin)
- Apache Maven 3.9.9

> [!TIP]
> Use [sdkman](https://sdkman.io/), managing different JDK versions the easy way
